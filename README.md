# M1 - Applied Math in Games: Projectile Game

A simple Unity 3D project demonstrating projectile motion using physics and basic C# scripting.

## Features
- Cube object that shoots projectiles (space or left click)
- Rigidbody physics for realistic motion
- Adjustable launch force and angle
- Simple, clean environment (plane + cube + sphere)

## How to Play
1. Press Space or Left Mouse Button to shoot.
2. Observe projectile arc and motion.
